## Overview

Nexus is a production-grade e-commerce platform built with Next.js App Router, featuring server-side rendering for optimal SEO and performance.

## Key Features

- Stripe Checkout with webhooks for order fulfillment
- Dynamic product catalog with search and filtering
- Server Components for zero JS overhead on static content
- Optimistic UI updates with React Server Actions

## Technical Highlights

The checkout flow uses Stripe's Payment Intents API with webhook verification for reliable order processing, even under network failures.
