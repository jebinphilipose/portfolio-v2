## Overview

OpenDocs Engine is a blazing-fast static site generator purpose-built for technical documentation, compiled to WebAssembly for browser-side rendering.

## Key Features

- Sub-100ms build times for large documentation sites
- MDX support with custom component injection
- Automatic API reference generation from OpenAPI specs
- Full-text search via WASM-compiled index

## Technical Highlights

The core parsing and rendering pipeline is written in Rust, then compiled to WebAssembly. This means the same engine runs both server-side (for builds) and client-side (for live preview), sharing 100% of the code.
