## Overview

Synth Analytics is a real-time data visualization platform that ingests and processes millions of events daily using a GraphQL subscriptions architecture.

## Key Features

- Real-time dashboard updates via GraphQL subscriptions
- D3.js custom visualizations with smooth transitions
- Time-series data with configurable aggregation windows
- Role-based access control and multi-tenancy

## Technical Highlights

The backend uses a CQRS pattern to separate read and write models, enabling the platform to scale reads independently from the write-heavy ingestion pipeline.
