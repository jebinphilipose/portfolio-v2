## Why Tailwind for Design Systems?

Tailwind CSS provides the primitives to build any design system without imposing one on you. The key is in the configuration.

## Setting Up Your Token System

Define your tokens in `tailwind.config.ts`:

```ts
colors: {
  primary: '#135bec',
  background: {
    light: '#f6f6f8',
    dark: '#101622',
  }
}
```

## Typography Scale

A consistent type scale is the foundation of any great design system. Use `font-size` and `line-height` in tandem.

## Dark Mode

Use the `class` strategy for full control over when dark mode activates.
