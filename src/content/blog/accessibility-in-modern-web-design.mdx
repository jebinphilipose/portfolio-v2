## Why Accessibility Matters

Accessibility isn't a feature — it's a foundation. Building accessible applications benefits everyone, not just users with disabilities.

## ARIA Patterns

ARIA (Accessible Rich Internet Applications) attributes bridge the gap between custom components and assistive technologies.

Key patterns to know:
- `aria-label` for elements without visible text
- `aria-expanded` for disclosure widgets
- `aria-live` for dynamic content updates
- `role` for semantic meaning

## Keyboard Navigation

Every interactive element must be reachable and operable via keyboard. Test your application without a mouse.

## Color Contrast

WCAG 2.1 AA requires a contrast ratio of at least 4.5:1 for normal text. Use tools like the WebAIM Contrast Checker to verify.
